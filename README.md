# Language School Project

Экзаменационный проект по дисциплине "Основы веб-технологий"

## Описание проекта

Веб-сайт языковой школы для изучения иностранных языков, разработанный в соответствии с экзаменационным заданием.

## Функциональность

- Главная страница с информацией о школе
- Блок преимуществ изучения языков
- Каталог языковых курсов с поиском и фильтрацией
- Система заявок на обучение
- Личный кабинет для управления заявками
- Адаптивный дизайн для мобильных устройств

## Используемые технологии

- HTML5
- CSS3
- Bootstrap 5.3
- JavaScript (ES6+)
- Fetch API для работы с AJAX
- Git для контроля версий

## API Интеграция

Проект использует предоставленный API по адресу:
`http://exam-api-courses.std-900.ist.mospolytech.ru/`

Поддерживаемые endpoints:
- `GET /api/courses` - список курсов
- `GET /api/courses/{id}` - информация о курсе
- `GET /api/tutors` - список репетиторов
- `GET /api/orders` - список заявок пользователя

## Особенности реализации

1. **Пагинация**: Реализована на клиентской стороне (10 курсов на страницу, 5 заявок на страницу)
2. **Поиск**: Реальный поиск по названию и фильтрация по уровню курса
3. **Адаптивность**: Полностью адаптивный дизайн для мобильных устройств
4. **Уведомления**: Система всплывающих уведомлений с автоматическим скрытием
5. **Обработка ошибок**: Грамотная обработка ошибок API и сетевых проблем

## Установка и запуск

1. Клонируйте репозиторий
2. Откройте файл `index.html` в браузере
3. Для работы требуется подключение к интернету (для загрузки Bootstrap и доступа к API)
